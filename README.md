# Cricket Academy 🏏

**Learn Cricket Like a Pro**

A professional cricket academy website featuring tutorials and training programs for all skill levels.

## Features

- 🏏 **Batting Tutorials** - Master the art of batting with techniques and shot selection
- 🎯 **Bowling Tutorials** - Learn different bowling styles and strategies
- 👥 **Fielding Tutorials** - Improve catching, throwing, and positioning
- 💪 **Fitness Training** - Build strength and endurance with cricket-specific programs

## Website Sections

1. **Home** - Hero section with call-to-action
2. **Courses** - Showcase of all training programs
3. **About** - Why choose our academy
4. **Contact** - Get in touch with us

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern responsive design with animations
- **JavaScript** - Interactive features and smooth scrolling

## Features

✨ Responsive design - Works on desktop, tablet, and mobile
✨ Smooth animations and transitions
✨ Modern gradient design
✨ Contact form functionality
✨ Sticky navigation bar
✨ Intersection Observer for scroll animations

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's a static website!

## File Structure

```
The-Cricket-Club/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # Interactive features
└── README.md       # Documentation
```

## Customization

You can easily customize:
- Colors in `styles.css` (gradient colors, button colors)
- Course content in `index.html`
- Contact form behavior in `script.js`
- Font and spacing values

## Deployment

To deploy this website:
1. GitHub Pages (free) - Enable in repository settings
2. Netlify (free) - Drag and drop your files
3. Vercel (free) - Connect your GitHub repository
4. Any web hosting service
